## Development
To run a local WOLF-CHALLENGE development server and view the application in your browser:

#### Step 1. Clone repo
`git clone https://github.com/vickcharles/wolf-challenge.git`

#### Step 2. Install dependecies
`npm install`

#### Step 3. Start server
`npm start`
